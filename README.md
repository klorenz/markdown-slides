# Markdown Slides

Markdown Slides lets you manage a set of slides and show it.


## Install

Install markdown-slides globally with **npm**

```bash
$ npm install -g markdown-slides
```


## Showing Slides

You can show your slides running

```bash
$ markdown-slides show
```
